# ⚔️ Elemental Survivor

PoE/Noita 스타일의 **메커니즘 조합** 서바이벌 게임

## 🎮 게임 소개

**Elemental Survivor**는 기본 주문에 모디파이어를 조합하여 무한한 빌드를 설계하는 웹 브라우저 게임입니다.

- 🔥❄️⚡🧪💨🪨 **6가지 기본 주문**
- 🔱🗡️🎯⛓️🔀💥🌀🪃 **12가지 모디파이어**
- 주문 + 모디파이어 조합으로 메커니즘이 변형됩니다!

### 예시 빌드

- **유도 폭격기**: 화염구 + 다중발사 + 유도 + 폭발
- **분열 번개**: 전격 + 분열 + 분열 + 연쇄
- **궤도 방패**: 빙결탄 + 궤도 + 다중발사
- **기관총**: 전격 + 속사 + 속사 + 관통

## 🕹️ 조작법

- **WASD / 방향키** - 이동
- **마우스 좌클릭** - 주문 발사 (커서 방향)
- **레벨업 시** - 새 주문 / 모디파이어 / 패시브 선택

## 🚀 실행 방법

```bash
# 로컬 서버 실행 (ES 모듈은 file:// 에서 동작하지 않음)
python3 -m http.server 8080

# 브라우저에서 http://localhost:8080 접속
```

## 🎯 핵심 메커니즘

### 주문 슬롯 시스템

```
SpellSlot = 기본 주문 + 모디파이어[]
```

모디파이어는 투사체의 **발사(cast)**, **비행(flight)**, **명중(hit)** 단계에 영향을 줍니다.

- **발사 단계**: 다중발사, 속사, 강화
- **비행 단계**: 유도, 궤도, 부메랑, 궤적
- **명중 단계**: 관통, 연쇄, 분열, 폭발, 연발
